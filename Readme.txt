+Nội dung:
Chương trình vẽ 5 loại hình cơ bản:
1. Đường thẳng (line)
2. Hình chữ nhật (rectangle). Nếu giữ phím Shift sẽ vẽ hình vuông (Square)
3. Hình Ellipse. Dùng hàm Ellipse. Nếu giữ phím Shift sẽ vẽ hình tròn (Circle)
Sử dụng GDI+ để vẽ và sử dụng Ribbon để tạo giao diện. Có thể xem ở chế độ preview

+Các Luồng sự kiện chính
1. Chọn loại hình cần vẽ cần vẽ từ menu, kéo thả để vẽ hình đã chọn.
2. Nếu không chọn, hình mặc định sẽ là LINE.
